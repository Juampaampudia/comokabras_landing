# Landing Page ComoKabras

Esta es la réplica del diseño "Alpins" adaptada para **ComoKabras**, siguiendo la estructura solicitada y utilizando el contenido proporcionado.

## Estructura del Proyecto

- `index.html`: Estructura principal de la página.
- `css/styles.css`: Estilos visuales (Diseño Dark/Light, Glassmorphism, Tipografía).
- `js/main.js`: Lógica para el menú móvil y efectos de scroll.

## Cómo ver la página

Para ver la página en tu ordenador local:

1.  Abre la carpeta en VS Code.
2.  Si tienes la extensión "Live Server", haz clic en "Go Live".
3.  O abre directamente el archivo `index.html` en tu navegador (aunque algunos iconos o fuentes podrían tardar en cargar sin un servidor local).

## Personalización

-   **Imágenes**: Las imágenes actuales son de Unsplash. Para usar tus propias fotos, guárdalas en una carpeta `assets/img` y actualiza las rutas en `index.html` y `css/styles.css` (para el fondo del hero).
-   **Textos**: Todo el contenido está en `index.html` y se puede editar fácilmente.
